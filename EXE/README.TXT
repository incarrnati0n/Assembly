  To install TECH Help!: Put the disk in drive A (or B), then type...

    A:INSTALL潰   (or B:INSTALL潰)

  When installed, use:

    XVIEW          to browse through the Help! text.
    XVIEW topic    (example: XVIEW ansi) to go directly to a topic.

    XVIEW /?       or
    XVIEW xview    for information on the XVIEW command options.

 If the default screen colors make the text hard to read, try:

   XVIEW /B

 or, in TECH Help!, select "Custom Colors" from the Configure/Colors menu.
 Remember to select "Save Setup" from the Configure menu.
